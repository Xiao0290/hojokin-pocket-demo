A single chat message bubble for the expert-messaging thread.

```jsx
<ChatBubble variant="system">事業計画書を共有しました</ChatBubble>
<ChatBubble from="me">ご確認お願いします。</ChatBubble>
<ChatBubble from="them">承知しました。</ChatBubble>
```

- `me` = brand blue, right-aligned, notched bottom-right; `them` = grey, left, notched bottom-left
- `variant="system"` = centered grey note (no bubble)
